Architecture decisions 7 invariants

AD-1
One app, one language
Next.js 16 + TypeScript, Route Handlers. Prevents a needless second service.
AD-2
Scorer is pure; UI only renders
All math in one server module. Prevents two sources of truth drifting.
AD-3
Join offline, read at runtime
Pipeline emits committed data.json. Prevents in-request data joins.
AD-4
LLM explains, never decides
Fed the computed score. Prevents the thin-wrapper failure.
AD-5
Free-text is data, not instructions
Delimited block, length cap, rate limit, server-only secrets. Prevents prompt injection + cost abuse.
AD-6
Stateless
No accounts, no DB, no persistence. Prevents unneeded data-governance surface.
AD-7
AWS Amplify + Route 53
Managed SSR, auto HTTPS. Prevents ops sprawl + insecure config.
